MLB Wins Prediction Dataset

This dataset contains various baseball statistics from the 2014 Major League Baseball season, used to predict the number of wins for the 2015 season.

Feature Description
WNumber of Wins
RRuns Scored
ABAt Bats
HHits
2BDoubles
3BTriples
HRHome Runs
BBBase on Balls (Walks)
SOStrikeouts
SBStolen Bases
RARuns Allowed
EREarned Runs
ERAEarned Run Average
CGComplete Games
SHOShutouts
SVSaves
EErrors

For more details on baseball statistics, visit MLB Stats.
